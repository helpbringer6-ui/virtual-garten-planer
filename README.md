# Virtueller Garten Planer

Willkommen bei Virtueller Garten Planer, einem innovativen Augmented-Reality-Tool zur Gestaltung und Visualisierung von Gärten.

## Vorteile

- **Echtzeit-Visualisierung**: Sehen Sie, wie Pflanzen in Ihrem Garten aussehen werden, bevor Sie sie kaufen.
- **Interaktive Anpassungen**: Passen Sie die Position und Anordnung der Pflanzen nach Ihren Wünschen an.
- **Pflanzeninformationen**: Erhalten Sie umfassende Informationen zu jeder Pflanze.

## Installation

1. Klonen Sie das Repository:
   ```bash
   git clone https://github.com/IhrBenutzername/virtueller-garten-planer.git
   ```
2. Installieren Sie die erforderlichen Abhängigkeiten:
   ```bash
   npm install
   ```
3. Starten Sie die Anwendung:
   ```bash
   npm start
   ```

## Nutzung

1. Öffnen Sie die App auf Ihrem Android- oder iOS-Gerät.
2. Scannen Sie den Bereich, in dem Sie planen möchten.
3. Wählen Sie Pflanzen aus der Bibliothek aus und platzieren Sie sie in Ihrer Umgebung.
